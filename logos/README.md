# BetterTunnels Logo Pack

A simple, modern logo direction for **BetterTunnels** — a secure developer tunnelling product similar to Cloudflare Tunnels/ngrok, with a cleaner security + tunnel motif.

## Included

### SVG
- `bettertunnels-horizontal-light.svg`
- `bettertunnels-horizontal-dark.svg`
- `bettertunnels-square-light.svg`
- `bettertunnels-square-dark.svg`
- `bettertunnels-mark-transparent.svg`
- `bettertunnels-mark-square-light.svg`
- `bettertunnels-mark-square-dark.svg`

### PNG

## Logo concept

The mark combines:
- a tunnel/portal arch
- a road/path entering the tunnel
- a shield-like app icon shape through the square variant

It is intentionally simpler than the previous concept: no circuit lines, no heavy emblem, no complex gradients outside the core blue mark.

## Colours

- Primary blue: `#0B6CFF`
- Deep blue: `#0054E8`
- Navy text: `#071A3A`
- Dark background: `#071225`
- White: `#FFFFFF`

## Recommended usage

- Use **horizontal light** on white/light backgrounds.
- Use **horizontal dark** on dark/navy backgrounds.
- Use **square light/dark** for app cards, docs tiles, social avatars, and product listing images.
- Use the **mark square** for favicons and compact app icons.

## Notes

The SVG files use system sans-serif fonts for the wordmark. For a production lockup, convert the wordmark to outlines in Figma/Illustrator/Inkscape after final approval.
